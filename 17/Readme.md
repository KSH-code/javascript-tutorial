## Strict mode
Strict mode는 엄격 모드다. 고의적으로 일반 코드와 다른 시멘틱을 가지고 있다.<br>
엄격 모드를 지원하지 않는 부라우저에서는 엄격 모드의 코드가 다른 방식으로 동작할 것이다.<br>
비-엄격 모드와 엄격 모드는 서로 공존이 가능하다.<br>
