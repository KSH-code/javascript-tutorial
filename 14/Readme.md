# 2017-08-31
## 개요
JS는 유형 및 연산자, 표준 내장 객체 메소드가 있는 다중 패러다임, 동적 언어다.<br>
구문은 JAVA, C 언어를 기반으로 한다.<br>
JS는 Class 대신 객체 프로토 타입을 사용해 객체 지향 프로그래밍을 지원한다.<br>
함수형 프로그래밍도 지원한다.<br>
타입 구조 : 
* 수
* 문자열
* 부울
* 기호
* 객체
    * 함수
    * 배열
    * 날짜
    * 정규식
* null
* undefined
그리고 오류 타입이 내장 돼있다.<br>

## 변수
JS의 변수는 let,const,var 키워드로 선언된다.<br>
* let : 블록 유효 범위 변수
* const : 값이 변경되지 않는 변수
* var : 함수 범위
